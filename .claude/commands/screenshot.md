# Screenshot Viewer

Δες τα τελευταία στιγμιότυπα οθόνης του χρήστη.

## Οδηγίες

Ο αριθμός που δίνεται μετά το `/screenshot` είναι πόσα screenshots να δείξεις (default: 1).

Παραδείγματα:
- `/screenshot` → δείξε το τελευταίο 1
- `/screenshot 3` → δείξε τα τελευταία 3

## Βήματα

1. Βρες τα τελευταία N screenshots από:
   `C:\Users\alexa\OneDrive\Εικόνες\Στιγμιότυπα οθόνης\`
   Ταξινόμησε κατά `LastWriteTime` φθίνουσα και πάρε τα πρώτα N.

2. Διάβασε και δείξε κάθε εικόνα με το Read tool.

3. Σχολίασε σύντομα τι βλέπεις σε κάθε screenshot.
